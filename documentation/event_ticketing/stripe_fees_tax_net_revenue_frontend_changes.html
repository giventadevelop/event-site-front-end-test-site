1. Fix Purchase Flow Calculations

1.1 Problem Statement

Issue: total_amount and final_amount are being set to the same value during ticket purchase, indicating calculations are not being properly applied.

Expected Behavior:

  • total_amount = Sum of base ticket prices (before discounts, fees, taxes)
  • final_amount = total_amount - discount_amount + platform_fee_amount + tax_amount

2. Stripe Customer Portal - Desktop PRB Buttons

✅ YES - Customer Portal Can Show Desktop PRB Buttons (When Updating Payment Methods)

Desktop PRB Buttons Available in Customer Portal:
  • Google Pay Button: ✅ May appear when updating payment method on Chrome/Edge/Firefox (Desktop)
  • Apple Pay Button: ✅ May appear when updating payment method on Safari (macOS)
  • Conditional: PRB buttons appear based on browser support and customer's saved payment methods
  • Automatic Detection: Stripe handles detection automatically - no code needed

Note: Customer Portal is primarily for subscription management, not initial payments. PRB buttons appear when customers update their payment methods, not during initial checkout.

Use Case

Customer Portal shows PRB buttons when:

  • Customer wants to update their payment method
  • Customer needs to add a new payment method
  • Customer's current payment method failed

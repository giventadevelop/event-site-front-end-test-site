Navigation Guide

This section details how to navigate to all the QR scanning, check-in analytics, and sales analytics pages and functionality.

Primary Navigation Method: Admin Menu Dropdown

The easiest way to access all QR scanning and analytics features is through the Admin menu dropdown in the main navigation header.

Accessing the Admin Menu

  1. Log in to the application with an admin account
  2. Look for the "Admin" menu item in the top navigation bar (it appears with a dropdown arrow)
  3. Hover over or click the "Admin" menu item to reveal the dropdown submenu
  4. Select the desired page from the dropdown menu

Available Links in Admin Menu Dropdown:

  • QR Scanner - Links to /admin/qr-scanner

Workflow Navigation: QR Scanner to Check-In

QR Code Scanning and Check-In Flow

  1. Start at QR Scanner: Navigate to /admin/qr-scanner using the Admin menu dropdown
  2. Scan QR Code: Use the camera to scan a ticket QR code
    • The scanner will automatically verify the QR code
    • If valid, it displays transaction details
  3. Automatic Redirect: After a successful scan (2-second delay), the system automatically redirects to the check-in page
    • Check-In Page URL Pattern: /admin/qrcode-scan/tickets/events/{eventId}/transactions/{transactionId}
    • Example: /admin/qrcode-scan/tickets/events/6901/transactions/5851
  4. Complete Check-In: On the check-in page, you can:
    • View ticket and attendee details
    • Update check-in status
    • Add number of guests
    • Add notes or comments
    • Use "Scan Another QR Code" button to return to the QR scanner

Navigation from Check-In Form

Scan Another QR Code Button

  • After completing a check-in, the check-in form includes a "Scan Another QR Code" button
  • Clicking this button redirects back to /admin/qr-scanner
  • This allows for continuous scanning and check-in workflow without manually navigating back to the scanner

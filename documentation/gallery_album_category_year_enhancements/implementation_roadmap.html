9. Test checklist (QA)

  • GET gallery-albums returns albumYear and nested galleryCategory
  • Admin saves album with category + year; reload shows values
  • Public card shows “Ecumenical Visits” pill and “2019” year
  • Photo count badge correct for album with media
  • Album without category/year — card renders without errors
  • MOSC redesign gallery loads from API (fallback if API down)
  • View Album navigates or opens slideshow per surface
  • Multi-tenant: category slug unique per tenant only

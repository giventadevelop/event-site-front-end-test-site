2. Entity / mapper changes

GalleryAlbum.java checklist
  • @Column(name = "event_date_start") private LocalDate eventDateStart;
  • @Column(name = "event_date_end") private LocalDate eventDateEnd;
  • @Column(name = "event_location", length = 256) private String eventLocation;
  • Map in GalleryAlbumMapper (MapStruct) — bidirectional
  • Liquibase changelog applied before entity compile

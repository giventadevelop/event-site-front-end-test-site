8. Navigation flow (v2.1)

  1. Event page → Competitions button → Hub
  2. Hub card → Competition detail (new)
  3. Detail → Register → Wizard (pre-selected competition)
  4. GROUP path → Team roster step → Payment
  5. Admin list → Edit competition → Guardrails + discipline
  6. Admin results → Publish → Public winners podium section

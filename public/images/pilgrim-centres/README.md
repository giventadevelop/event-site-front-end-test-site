This directory contains images for the Pilgrim Centres page.

1. `thiruvithamcode.jpg` - Thiruvithamcode Arappally

3. `niranam.jpg` - St. Mary's Orthodox Syrian Church, Niranam
4. `arthat.jpg` - Arthat St. Mary's Cathedral, Kunnamkulam

7. `koonan-kurishu.jpg` - Koonan Kurishu Pilgrim Centre, Mattancherry
8. `old-seminary.jpg` - Old Seminary (Pazhaya Seminary), Kottayam

10. `cheriapally.jpg` - Kottayam Cheriapally (St. Mary's Orthodox Syrian Church)
11. `kallooppara.jpg` - St. Mary's Orthodox Church, Kallooppara
12. `chandanapally.jpg` - St. George Orthodox Church, Chandanapally

## Image Guidelines

- **Format**: JPEG or WebP
- **Recommended Size**: 800x600px (4:3 aspect ratio)
- **Optimization**: Compress images to reduce file size while maintaining quality
- **Content**: Professional photos of the churches/centres showing their architectural beauty
- **Copyright**: Ensure proper rights to use the images

## Placeholder Images

Dynamic Event Test Suite - Separation Guide

Date: January 7, 2026

Purpose: Separate dynamic event discovery and testing into its own script for better maintainability and independent execution.

Overview

The dynamic event testing logic has been extracted from the main admin test suite into a separate, dedicated script. This allows:

  • Independent execution of dynamic event tests
  • Separate test reports for dynamic event pages
  • Easier debugging of event discovery logic
  • Faster execution of static admin page tests
  • Better organization and maintainability

File Structure

File Purpose Report Output

Dynamic Event Pages Tested

The dynamic event test suite tests the following pages for each discovered event:

  1. Event Details Page - /admin/events/{eventId}
  2. Event Media Upload Page - /admin/events/{eventId}/media
  3. Event Media List Page - /admin/events/{eventId}/media/list
  4. Event Tickets List Page - /admin/events/{eventId}/tickets/list
  5. Event Ticket Types List Page - /admin/events/{eventId}/ticket-types/list

Event Discovery Process

  1. Navigate to /admin/manage-events
  2. Wait 8 seconds for page to fully load
  3. Check for events table using exact XPath
  4. If no events:
    • Find toggle button using exact XPath
    • Click toggle to switch Future/Past events
    • Wait 8 seconds for data reload
    • Check again for events
  5. Extract event IDs from "Upload Media Files" links (column 9)
  6. Select up to 3 random events for testing
  7. Test all dynamic event pages for selected events

Troubleshooting

No Events Found

If the dynamic test suite can't find events:

  1. Check that events exist in the system
  2. Verify you can access /admin/manage-events manually
  3. Try toggling between Future/Past events manually
  4. Check the console logs for page structure debugging info
  5. Verify the XPath paths match your page structure

Event Discovery Fails

The script includes detailed debugging output:

  • Page structure information (body children, div counts, etc.)
  • Table detection status
  • Toggle button detection status
  • Event ID extraction details

Review the console output to see exactly what's happening at each step.

Updates types, shared date formatter, admin album forms, and public cards so gallery UI matches the legacy date line (full date, optional range, optional location).

1. Type definitions — src/types/index.ts

  // 3. Single day → "November 25, 2015"

  • Send ISO date strings (YYYY-MM-DD) only — no time component
  • Empty strings → omit or send null to clear location/dates

5. Static fallback (moscStaticAlbums)

When API albums unavailable, static slice should pass date from MOSC_STATIC_GALLERY_ALBUMS as a display override OR parse client-side until backfill completes. Recommended: add optional eventDateDisplay?: string prop for static-only fallback using raw album.date.

DB Schema Requirements

Table event_attendee_attachment includes:

  • id, tenant_id, attendee_id, event_id, title, description
  • file_url, content_type, file_size, original_filename
  • storage_type, event_media_type, is_public, uploaded_by_id
  • created_at, updated_at
  • FKs to event_attendee(id) and event_details(id)
  • Index on (attendee_id, created_at)

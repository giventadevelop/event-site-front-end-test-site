6. Security & tenancy

  • JWT Bearer; tenant from TenantContextFilter and X-Tenant-ID
  • Guardian (YOUTH): mutate only own child participants and registrations
  • Competitor (ADULT): mutate only self-linked participant rows
  • Team captain: group registrations where registered_by_user_profile_id matches
  • Admin: catalog CRUD, results entry, publish, winner media linkage
  • Public read: published results, active competitions, content blocks (align with proxy public routes)

Prompt: SquadRosterSection

Build SquadRosterSection (client component):
- Dark full-width section (#0f172a style), eyebrow from team_groups.sectionLabel, H2 from headline or name.
- Fetch GET /api/proxy/team-groups?isActive.equals=true&sort=displayOrder,asc (relative URL).
- Use first group; fetch members with teamGroupId.equals and sort=priorityOrder,asc.
- Horizontal scroll carousel with prev/next; mobile swipe.
- CTA pill from ctaLabel/ctaHref (default /team/{slug}).
- Return null if no group or no members. object-contain portraits, dark letterbox bg.

Prompt: Admin TeamMemberForm

Admin form for team_members:
- Required: teamGroupId select, firstName, lastName.
- If group.teamType === MUSIC: show instrument, vocalRole; hide jerseyNumber.
- Else: show jerseyNumber, position.
- expertise as dynamic string list → JSON.stringify on save.
- Optional userProfileId number field; portrait via DragDropImageUpload + upload proxy.
- priorityOrder tooltip: lower = higher rank in carousel.

4. Root cause: wrong categorySlug in manifest

_scrape-downloads-deep.mjs assigns every scraped file to categorySlug: "mosc-in-downloads". Upload scripts use this slug (or mirror folder name), so Malankara PDFs never appear when filtering by Malankara Association 2022.

// scripts/mosc-in-migration/_scrape-downloads-deep.mjs (line 22)
const DEFAULT_CATEGORY_SLUG = 'mosc-in-downloads';

// Every manifest item gets:
categorySlug: DEFAULT_CATEGORY_SLUG

6. Recommended closure (Phase 1)

  1. Build or patch manifest with correct categorySlug per legacy page mapping (§2).
  2. Fetch PDFs to folder structure: {MOSC_DOWNLOAD_ROOT}/{categorySlug}/{year}/ or mirror tree with category as top folder.
  3. Run upload-mirror-to-official-docs.mjs with correct tenant and MOSC_AUTO_CREATE_CATEGORIES=true if needed.
  4. Verify on http://localhost:3000/mosc-redesign/downloads with category filter Malankara Association 2022.

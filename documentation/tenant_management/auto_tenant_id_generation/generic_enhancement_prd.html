Auto Tenant ID Generation — Enhancement PRD Index

Project: event-site-manager  |  Version: 1.0  |  Date: June 19, 2026

When creating a new tenant organization, admins supply a name prefix (e.g. ford_motors). The system assigns the next global sequence number from existing tenant_organization.tenant_id values and produces IDs like ford_motors_10000. If no sequenced IDs exist yet, numbering starts at 00001.

ID format rules

  • Pattern: {prefix}_{sequence} — prefix lowercase slug; sequence is numeric suffix after the last underscore.
  • Padding: minimum 5 digits (00001, 10000).
  • Sequence scope: global max of trailing numbers across all tenant IDs (not per-prefix).
  • First tenant: {prefix}_00001 when no existing ID matches *_NNNNN.

Workflow note: Tenant IDs are created on Organizations → New Organization. Tenant Settings creation selects an existing organization; it does not mint a new tenant ID.

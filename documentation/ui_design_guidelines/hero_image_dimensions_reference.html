Hero Section — Image Dimensions Reference

Use these dimensions when creating or uploading images for each hero section.

Section 1 — Left Panel (Branding Image)

CSS: object-fit: contain — Image is ALWAYS fully visible, no cutoff on any side.
Frame: aspect-ratio: 5/6 — The frame auto-sizes to hug the image tightly.
PropertyValue
Aspect Ratio5:6 portrait (0.83:1)
Recommended Size1000 × 1200 px
Minimum Size600 × 720 px
Fit Behaviourobject-contain — full image, no cropping
Safe ZoneEntire image is visible — use the full canvas

Since the frame matches the image ratio (5:6), a 5:6 image fills the frame perfectly with zero gaps and zero cropping. Logos and edge text are fully visible.

Section 2 — Right Panel (Slideshow / Event Images)

CSS: object-fit: cover — Image FILLS the container at every viewport width.
Why cover? The container width changes dramatically with the browser (from 459px on tablet to 1254px on large desktop), but the height stays fixed. The ratio swings from 1.55:1 to 3.56:1. No single image can match all sizes, so object-cover fills the space and crops edges minimally.

Quick Summary

Section 1: Provide 5:6 portrait images (1000×1200px). Full image always visible.
Section 2: Provide 5:2 wide landscape images (2000×800px). Keep important content in center 60%.
Section 3: Provide 5:1 wide banner images (1000×200px).

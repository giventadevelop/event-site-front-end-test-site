3. Strapi API contract

List members (tenant-scoped)
GET /api/holy-synod-members?filters[tenant][tenantId][$eq]=tenant_demo_002&populate[0]=image&sort=order:asc,name:asc

Detail: GET /api/holy-synod-members?filters[slug][$eq]={slug}&filters[tenant][tenantId][$eq]=...

Key fields used by CMS UI

name, slug, title, excerpt, body, image, order, tenant

4. UI components reused

  • SyroPageBanner — breadcrumbFrom: 'holy-synod-cms'
  • MoscHubCardMedia — hub grid cards on list page
  • QuickLinks — Calendar quick links in layout
  • export const dynamic = 'force-dynamic' on list + detail pages

5. Environment

  • STRAPI_URL or NEXT_PUBLIC_STRAPI_URL — Strapi base (e.g. http://localhost:1337)
  • NEXT_PUBLIC_TENANT_ID — must match imported tenant (e.g. tenant_demo_002)

Calendar CMS Menu Items – Front-end PRD Index

Documentation for adding Calendar submenu entries in mosc-temp that mirror static MOSC redesign pages with Strapi-backed -cms routes. Strapi content lives in C:\project_workspace\strapi-editorial-template.

Scope: Documentation only — no implementation in this folder.

Reference pattern: Holy Synod CMS (already wired in mosc-temp).

Menu config file: src/components/mosc-redesign/calendarNav.ts

Strapi import runbooks: strapi-editorial-template/documentation/front_end_based_collection_types_import/

Architecture (static → Strapi → CMS route)

  1. Static source — Hard-coded TSX under src/app/mosc-redesign/(syro)/{section}/ (legacy, preserved).
  2. Strapi — Collection type in strapi-editorial-template; import scripts populate entries from static TSX + images.
  3. CMS route — New /mosc-redesign/{section}-cms list + [slug] detail pages fetch from Strapi at request time.
  4. Calendar menu — Add { label: '{Section} CMS', href: '/mosc-redesign/{section}-cms' } to CALENDAR_MENU_ITEMS.

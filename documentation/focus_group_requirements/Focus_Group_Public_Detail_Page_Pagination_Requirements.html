1. Overview

This document defines the requirements for adding pagination and event filtering (upcoming/past toggle) to the public-facing Focus Group detail page. The page currently displays all events in a grid without pagination. This enhancement will improve performance and user experience for focus groups with many events.

⚠️ Important: This document contains requirements only. Do NOT implement until explicitly requested. Focus on understanding the requirements and creating a comprehensive implementation plan.

4.2 Server Actions

File:

Create: src/app/focus-groups/[slug]/ApiServerActions.ts

Required Functions:

  • fetchFocusGroupEventsServer(focusGroupId: number, page: number, pageSize: number, showPastEvents: boolean)
    • Builds appropriate query based on showPastEvents
    • Uses focusGroupId.equals filter
    • Adds date filter (upcoming or past)
    • Adds pagination parameters
    • Uses fetchWithJwtRetry from @/lib/proxyHandler
    • Returns { events: EventDetailsDTO[], totalCount: number }
    • Reads X-Total-Count header

Implementation Pattern:

import { fetchWithJwtRetry } from '@/lib/proxyHandler';
import { getAppUrl } from '@/lib/env';
import type { EventDetailsDTO } from '@/types';

const API_BASE_URL = process.env.NEXT_PUBLIC_API_BASE_URL;

export async function fetchFocusGroupEventsServer(
  focusGroupId: number,
  page: number,
  pageSize: number,
  showPastEvents: boolean
): Promise<{ events: EventDetailsDTO[]; totalCount: number }> {
  const baseUrl = getAppUrl();
  const today = new Date().toISOString().split('T')[0];

  const params = new URLSearchParams({
    'focusGroupId.equals': focusGroupId.toString(),
    'isActive.equals': 'true',
    page: page.toString(),
    size: pageSize.toString(),
  });

  if (showPastEvents) {
    params.append('endDate.lessThan', today);
    params.append('sort', 'startDate,desc');
  } else {
    params.append('startDate.greaterThanOrEqual', today);
    params.append('sort', 'startDate,asc');
  }

  const url = `${API_BASE_URL}/api/event-details?${params.toString()}`;
  const res = await fetchWithJwtRetry(url, { cache: 'no-store' });

  if (!res.ok) {
    throw new Error(`Failed to fetch events: ${res.statusText}`);
  }

  const totalCount = Number(res.headers.get('X-Total-Count')) || 0;
  const events = await res.json();

  return {
    events: Array.isArray(events) ? events : [],
    totalCount
  };
}

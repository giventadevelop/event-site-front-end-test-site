# AI Image Editor Prompt: Extract Unite India Logo for Header

Use this prompt when you upload `unite_india_logo.avif` to an AI image editor (e.g. Runway, Photopea, Leonardo, or similar) to get a header-sized logo with only the "giving hands" graphic, no text.

---

## Target use
- **Location:** Top-left of the site header, next to the "Unite India" text  
- **Current placeholder size:** 40×40 px (Tailwind `w-10 h-10`)  
- **Recommended export size:** 80×80 px or 96×96 px (2x for sharp display)  
- **Format:** PNG with transparent background, or same format as source  

---

## Copy-paste prompt (instruction / edit mode)

```
Using the attached image (unite_india_logo.avif):

1. Extract ONLY the "giving hands" icon — the upper graphic. It shows two stylized hands in white outline, cupped upward, with three minimalist human figures (white outline) inside: one smaller central figure and two larger figures to the sides. Do NOT include any text ("Unite India" or otherwise) that appears below this graphic.

2. Remove the black background: make it fully transparent so only the white outline (hands and figures) remains. The icon must read clearly on any header background.

3. Output a square image, 80×80 or 96×96 pixels, containing just this icon centered with some padding. Do not redraw or change the design — only crop/isolate the hands-and-figures portion and make the background transparent.

4. This image will be shown at 40×40 px in a website header next to the text "Unite India", so the icon must stay sharp and recognizable at that size.
```

---

## Shorter variant (minimal prompt)

```
From the attached image, extract only the "giving hands" graphic (two cupped hands with three small figures inside, white outline). Exclude all text. Make the black background transparent. Output a square 80×80 px image, centered, for use as a 40×40 px header icon.
```

---

## If the tool asks for “subject” or “output description”

```
Subject: Logo only — two hands in a heart or “giving hands” symbol, no typography.
Output: Square crop, 80×80 px, transparent background (replace black with transparency), ready for use as a 40×40 px header icon next to "Unite India".
```

---

## Notes for manual crop (e.g. Photopea / Figma)

- **Display size in header:** 40×40 px  
- **Export at:** 80×80 px or 96×96 px  
- **Crop to:** Only the hands/logo area; leave out any "Unite India" or tagline text  
- **Save as:** PNG (transparent) or WebP; name e.g. `unite_india_header_icon.png`  
- **Use in Header:** Place file in `public/images/` and reference as `/images/unite_india_header_icon.png`

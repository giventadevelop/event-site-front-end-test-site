6. Integration with frontend / webhook

Option A: Stripe webhook in mosc-temp calls batch job HTTP endpoint after team registration confirm (async).

Option B: Admin results publish action triggers winners-by-competition email job.

Option C: Admin registrations page "Export CSV" calls batch export job, emails download link or S3 presigned URL.
